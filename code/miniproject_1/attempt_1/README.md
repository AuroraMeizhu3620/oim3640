# Classroom SuperAssistant 1.0

This project is a Flask-based classroom management web application for managing a roster, creating a seating chart, tracking attendance and participation, and exporting grading data to Excel.

## Features

1. Student roster management with persistent JSON storage.
2. Classroom seating layout generation and seat assignment.
3. Session-by-session attendance and participation tracking.
4. Weighted scoring summary for attendance and participation.
5. Excel export for grading records.

## Run

1. Create a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Run `python app.py`.
4. Open the local Flask URL in your browser.
