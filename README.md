# My Projects
## 🧑‍🏫 Mini Project 1: Classroom SuperAssistant 1.0

**Description**  
Classroom SuperAssistant 1.0 is a Flask-based classroom management web application designed to help instructors manage seating charts, attendance, and participation more efficiently. The app allows an instructor to create a classroom seating layout, assign students to specific seats, record attendance and participation during each class session, store historical records, and export the data into an Excel file for grading.

## 📖 Mini Project 2: Quote Finder

**Description**  
Quote Finder is a text analysis tool that helps users search for specific words in a book or long text and return the full sentence where each word appears. If possible, the tool also returns the nearest page number so users can locate the quote in the original text more easily.

## 📰 Mini Project X: News Framing & Sentiment Analysis App
**Description**  
This project is a text analysis app that compares how different news platforms describe conflicts, protests, and political events through word choice. Users can paste article text into the app, and the program analyzes emotionally charged language, repeated framing words, and basic word-frequency patterns.

## What I Learned
- How to read and load long text files into Python so they can be processed programmatically
- How to clean raw text by handling spacing, line breaks, punctuation, and capitalization
- How to divide a long text into smaller searchable units, such as words, sentences, and possible page sections
- How to search for a specific word while still preserving the full sentence around it
- How to think about the difference between finding a word and finding useful context for that word
- How to use string methods and loops to scan through large amounts of text efficiently
- How to organize a program into smaller functions, such as loading text, searching text, compiling sentences, and identifying page numbers
- How text analysis can support literary research by making it easier to collect evidence for essays
- How repeated words can reveal themes, motifs, and symbols in a novel
- How difficult it can be to work with real-world text files because they are not always formatted consistently
- How page numbers are more complicated than expected because digital text files may not preserve the same pagination as printed books
- How to design a tool around a real academic problem instead of just writing code for practice
